# 🎉 欢迎使用 AutoAnki！

感谢选择AutoAnki - 你的智能Anki卡包生成助手！

## 🚀 三步快速开始

### 1️⃣ 安装依赖
```bash
npm install
```

### 2️⃣ 启动应用
```bash
npm start
# 或双击 start.bat (Windows) / start.sh (Mac/Linux)
```

### 3️⃣ 开始使用
打开浏览器访问：**http://localhost:3000**

## 💡 首次体验建议

### 🎯 零配置快速体验
不想配置API？没问题！

1. 打开应用
2. 粘贴 `examples/example-1-photosynthesis.txt` 中的内容
3. **按住 Shift 键**点击"制卡"按钮
4. 查看演示卡片
5. 点击"导出"下载.apkg文件

**就这么简单！** ✨

### 🔧 使用真实AI生成
想要个性化的卡片？

1. 复制 `.env.example` 为 `.env`
2. 填入你的API密钥（推荐使用DeepSeek，便宜好用）
3. 重启服务器
4. 享受AI生成的魔力！

## 💬 使用技巧

### ✏️ 材料准备
- 200-1000字最佳
- 结构清晰，分段明确
- 包含定义、例子、解释

### 🎯 卡片数量
- 短文本：5-10张
- 中等文本：10-20张
- 长文本：20-50张

### 🔍 质量检查
1. 生成后仔细检查
2. 编辑不准确的内容
3. 添加个人理解
4. 确保问答对应

## 🆘 遇到问题？

### 常见问题速查

**Q: 点击制卡没反应？**
- 检查是否输入了内容
- 查看浏览器控制台
- 尝试演示模式（Shift+点击）

**Q: 生成失败？**
- 检查网络连接
- 验证API配置
- 使用演示模式测试

**Q: 导出的文件打不开？**
- 确认文件扩展名是.apkg
- 使用最新版Anki
- 重新导出试试

### 获取帮助
- 查看文档（上方表格）
- 检查日志输出
- 在GitHub提Issue

## 🎓 学习建议

### 如何高效使用Anki
1. **每天复习** - 利用间隔重复算法
