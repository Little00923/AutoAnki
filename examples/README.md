# 示例学习材料

这个文件夹包含了一些示例学习材料，你可以用来测试AutoAnki的功能。

## 📚 示例文件列表

### 1. example-1-photosynthesis.txt
**主题**：生物学 - 光合作用  
**适合卡片数**：15-25张  
**内容**：光合作用的定义、过程、影响因素和意义

### 2. example-2-programming.txt
**主题**：编程 - JavaScript闭包  
**适合卡片数**：10-20张  
**内容**：闭包的概念、应用场景、优缺点和注意事项

### 3. example-3-history.txt
**主题**：历史 - 文艺复兴  
**适合卡片数**：20-30张  
**内容**：文艺复兴的时间、核心思想、主要成就和历史意义

## 🎯 使用方法

### 方法1：复制粘贴
1. 打开任意示例文件
2. 复制全部内容
3. 粘贴到AutoAnki的文本框中
4. 点击"制卡"

### 方法2：选择性使用
你也可以只复制文件中的部分内容来生成卡片，例如：
- 只复制某个章节
- 提取关键段落
- 组合多个来源的内容

## 💡 建议

### 卡片数量设置
- **短文本**（<300字）：5-10张
- **中等文本**（300-800字）：10-20张
- **长文本**（>800字）：20-50张

### 优化技巧
1. 确保文本有清晰的结构（标题、段落、要点）
2. 包含完整的定义和解释
3. 避免过于碎片化的信息
4. 可以添加例子和应用场景

## 🔨 创建自己的学习材料

你可以参考这些示例创建自己的学习材料文件。好的学习材料应该包含：

✅ **清晰的主题**：一个明确的学习目标  
✅ **系统的结构**：逻辑清晰的组织方式  
✅ **完整的解释**：定义、例子、应用  
✅ **关键概念**：核心知识点突出  
✅ **适当的长度**：200-1500字为宜  

## 📝 示例输出

使用这些材料生成的卡片类型包括：
- **基础问答卡**：测试核心定义
- **填空卡**：记忆关键术语
- **枚举卡**：列表和分类
- **理解卡**：因果关系和原理

1. 选择一个示例文件
2. 在AutoAnki中生成卡片
3. 预览和编辑结果
4. 导出并在Anki中学习
