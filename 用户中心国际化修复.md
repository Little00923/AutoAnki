## 使用方法

### 1. 刷新页面
访问用户中心并**强制刷新**（Ctrl+Shift+R 或 Cmd+Shift+R）：
```
http://localhost:3000/user-center.html
```

### 2. 切换语言
- 点击顶部的 **EN** 按钮 → 显示英文
- 点击顶部的 **中文** 按钮 → 显示中文

## 注意事项

### 如果页面仍显示中文
1. **强制刷新浏览器** (Ctrl+Shift+R)
2. **清除浏览器缓存**
3. 确认语言按钮状态（EN 应该高亮）

### 检查当前语言
在浏览器控制台输入：
```javascript
i18n.getCurrentLanguage()
```
应该返回：`"en"` 或 `"zh"`

### 手动切换语言
在浏览器控制台输入：
```javascript
i18n.switchLanguage('en')  // 切换到英文
i18n.switchLanguage('zh')  // 切换到中文
```
